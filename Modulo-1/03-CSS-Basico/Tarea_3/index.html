PSEUDOCLASE O PSEUDOELEMENTO

Estilos especiales dependiendo del contexto, posición o estado.

:root

La pseudo-clase de CSS :root selecciona el elemento raíz de un documento. Aplicado a HTML, :root representa el elemento <html> y es idéntico al selector html, excepto que su nivel de especificidad es mayor. Sirve para declarar variables CSS.
